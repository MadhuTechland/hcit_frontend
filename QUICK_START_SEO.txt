4. HOW THE SEO SYSTEM WORKS
───────────────────────────

When you navigate to a page:

1. Component loads → React Router matches the URL
2. SEO component detects path change → useEffect triggers
3. SEO component looks up page metadata from pageMetadata object
4. Meta tags are updated dynamically:
   - Title tag
   - Description meta tag
   - Keywords meta tag
   - Open Graph tags (og:title, og:image, etc.)
   - Twitter Card tags
   - Canonical URL
   - JSON-LD Structured Data

Result: Perfect SEO optimization for each page!

5. CUSTOMIZATION GUIDE
──────────────────────

TO UPDATE PAGE TITLES & DESCRIPTIONS:
1. Open src/components/SEO.jsx
2. Find the pageMetadata object
3. Update the title, description, keywords for any page
4. Changes take effect immediately!

Example:
  '/': {
    title: 'Your Custom Title',
    description: 'Your custom description',
    keywords: 'your, keywords, here',
    // ... rest of config
  }

TO ADD NEW PAGES:
1. Create new file in src/pages/NewPage.jsx
2. Add route to src/App.jsx
3. Add metadata entry to src/components/SEO.jsx pageMetadata
4. That's it! SEO is automatic!

TO CHANGE COMPANY INFO:
1. Update company name → src/components/Layout.jsx
2. Update contact info → src/components/Layout.jsx
3. Update structured data → src/components/SEO.jsx
4. Update favicon → public/favicon.ico or index.html
